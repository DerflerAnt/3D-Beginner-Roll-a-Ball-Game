# Unity Ball Runner Game - Третяк Олександр

Цей проект присвячений розробці повноцінного ігрового циклу в Unity, від створення фізичного керування персонажем до побудови архітектури переходів між рівнями та системи ігрових налаштувань. Основна мета роботи полягала у створенні стабільного білду з розгалуженим інтерфейсом та інтегрованим звуковим супроводом.

## Фізика руху та архітектура рівнів
В основі геймплею лежить скрипт PlayerController, що базується на фізичній взаємодії компонента Rigidbody. Керування м'ячем реалізовано через додавання сили (AddForce), що забезпечує інерційність та плавне прискорення. Окрему увагу приділено системі збору об'єктів та динамічному оновленню UI тексту, що відображає прогрес гравця в реальному часі.

Проект містить два повноцінних рівня з поступовим зростанням складності. На другому рівні додано ще одного ворожого персонажа (Enemy), що додає елемент небезпеки та змушує гравця діяти швидше. Взаємодія з ворогом реалізована через систему тригерів, що миттєво викликає панель програшу, зупиняючи ігровий час для забезпечення коректного User Experience.

## Система інтерфейсу та навігації
Головне меню гри розроблено як окрема сцена з функціоналом вибору рівнів та виходу з програми. Логіка навігації побудована на універсальному скрипті MenuManager, який використовує SceneManager для безшовного перемикання між ігровими станами. Кожен рівень завершується викликом EndGamePanel, де гравець має можливість перезапустити етап або повернутися в головне меню.

Для вирішення проблеми візуальних розривів картинки (Screen Tearing) у налаштуваннях проекту було активовано вертикальну синхронізацію (V-Sync). Це дозволило стабілізувати частоту кадрів відповідно до частоти оновлення монітора, зробивши рух камери та об'єктів максимально плавним навіть на високих швидкостях.

## Аудіо-система та динамічні налаштування
Особливістю проекту є реалізація панелі налаштувань звуку, що доступна безпосередньо під час ігрового процесу та в головному меню. За допомогою компонентів UI Slider та UI Toggle було створено інструменти для плавного регулювання гучності та повного вимкнення музики (Mute).

Технічна реалізація базується на передачі динамічних значений (Dynamic float/bool) зі слайдера безпосередньо в параметр volume компонента AudioSource через код. Такий підхід дозволив уникнути жорсткого програмування значень та надати гравцеві повний контроль над звуковим оточенням. Для стабільної роботи на різних рівнях була використана система префабів, що гарантує ідентичність налаштувань на кожній сцені.

## Технічні деталі керування
Весь функціонал гри повністю працездатний у фінальному білді. Керування в проекті здійснюється за допомогою стандартних осей вводу та інтерактивних елементів UI:
* Клавіші WASD або стрілочки - рух м'яча по ігровій зоні.
* Інтерактивні панелі (Win/GameOver) - автоматично активуються при завершенні етапу або зіткненні з ворогом, надаючи доступ до кнопок перезапуску рівня чи повернення в головне меню.
* Повзунок Volume - плавне регулювання гучності фонової музики в реальному часі.
* Перемикач Mute - миттєве вимкнення або увімкнення звукового супроводу.
* Кнопки Level 1/Level 2 - швидкий перехід до обраного етапу безпосередньо з головного меню.
