3D Beginner: Roll-a-Ball Game - Виконано.

У проекті було виконано базові вимоги туторіалу та реалізовано наступний функціонал для отримання максимального балу:
1. Базовий рух:
Переміщення гравця (кулі) повністю реалізовано за допомогою нової Unity Input System.
2. Візуал, енвайронмент та звук:
Створено повноцінні ігрові рівні з освітленням, перешкодами та об'єктами для збору.
Використано Particle Systems: налаштовано візуальні ефекти при підбиранні предметів, ефект вибуху гравця при програші та ефект розчинення ворогів.
Реалізовано звуковий супровід: фонова музика в меню, звуки підбирання предметів, ефекти програшу та перемоги.
3. Інтерфейси (UI):
Розроблено Головне меню (Main Menu) для старту гри, вибору рівня та виходу з програми.
Створено Меню кінця гри (End Game Menu), яке виводить фінальний рахунок (Score) та містить кнопки для рестарту (Restart) або повернення в головне меню при перемозі чи поразці.
4. Додаткові ігрові механіки:
Стрибок та Спринт (прискорення): Гравцеві реалізовано стрибок (з фізичною перевіркою землі) та можливість прискорення руху затисканням клавіші через Input System.
Вороги та система поразки: На рівнях присутні рухомі вороги, які реагують на гравця. При зіткненні з ворогом фіксується програш: гравець вибухає (партікли + звук), рух блокується і з'являється екран поразки.
Механіка зачистки рівня: Коли гравець збирає всі необхідні об'єкти (PickUps) на рівні, спрацьовує перемога: всі вороги на карті автоматично знищуються з візуальним ефектом розпаду.
